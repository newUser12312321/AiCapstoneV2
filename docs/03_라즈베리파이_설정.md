# 03. 라즈베리파이 설정 가이드 (Windows SSH 기준)

## 전제 조건

- Windows 데스크탑과 라즈베리파이가 **같은 공유기(LAN)**에 연결되어 있을 것
- 라즈베리파이에 Raspberry Pi OS (64-bit) 설치 완료

---

## 1단계 — 라즈베리파이 SSH 활성화

라즈베리파이에서 **최초 1회** 직접 실행:

```bash
sudo systemctl enable ssh
sudo systemctl start ssh

# IP 주소 확인 (이 숫자를 Windows에서 사용)
hostname -I
# 예: 192.168.0.25
```

또는 **Raspberry Pi Imager**로 OS 굽기 전 미리 활성화:
> 톱니바퀴(⚙️) → SSH 활성화 → 사용자명: pi / 비밀번호 설정

---

## 2단계 — Windows에서 SSH 접속

```powershell
# PowerShell 또는 Windows Terminal에서
ssh pi@192.168.0.25
# 비밀번호 입력
# 처음 접속 시: "Are you sure..." → yes 입력
```

### SSH 키 등록 (비밀번호 없이 접속)

```powershell
# Windows에서 SSH 키 생성 (없는 경우)
ssh-keygen -t ed25519 -C "windows-desktop"
# 엔터 3번 (기본 경로, 패스프레이즈 없음)

# 공개키를 라즈베리파이에 등록
type $env:USERPROFILE\.ssh\id_ed25519.pub | ssh pi@192.168.0.25 "mkdir -p ~/.ssh && cat >> ~/.ssh/authorized_keys"

# 이후 비밀번호 없이 접속 가능
ssh pi@192.168.0.25
```

## 3단계 — 라즈베리파이 시스템 패키지 설치

SSH 접속 후 라즈베리파이 터미널에서 실행:

```bash
sudo apt update && sudo apt upgrade -y

sudo apt install -y \
  python3-pip \
  python3-venv \
  v4l-utils \
  python3-gpiozero \
  python3-lgpio \
  git
```

## 6단계 — .env 환경변수 설정 (핵심)

```bash
cp .env.example .env
nano .env
```

수정 항목:

```dotenv
# ★ Windows 데스크탑 IP 주소로 변경 (ipconfig로 확인)
SERVER_BASE_URL=http://192.168.0.10:8080

# ★ 웹캠 장치 번호 (아래 명령으로 확인)
CAMERA_DEVICE_INDEX=0

# ★ 운영 환경으로 변경
ENVIRONMENT=production

# GPIO 핀 번호 (실제 배선에 맞게)
BUZZER_PIN=17
LED_RED_PIN=27
LED_GREEN_PIN=22
```

> `Ctrl+O` → Enter (저장), `Ctrl+X` (종료)

## 7단계 — 웹캠 확인

```bash
# 연결된 카메라 장치 확인
v4l2-ctl --list-devices
# C922 Pro Stream Webcam:
#     /dev/video0  ← CAMERA_DEVICE_INDEX=0

# 오토포커스 수동 제어 테스트
v4l2-ctl --device=/dev/video0 --set-ctrl=focus_auto=0
v4l2-ctl --device=/dev/video0 --set-ctrl=focus_absolute=30

# Python으로 캡처 테스트
python3 -c "
import cv2
cap = cv2.VideoCapture(0, cv2.CAP_V4L2)
ret, frame = cap.read()
print('캡처 결과:', '✅ 성공 -', frame.shape if ret else '❌ 실패')
cap.release()
"
```

## 8단계 — GPIO 하드웨어 배선

```
라즈베리파이 40핀 GPIO (BCM 번호 기준)
┌──────────────────────────────────────────┐
│  GPIO 17 (11번 핀) ─[직결]─ 부저(+)       │
│  GPIO 27 (13번 핀) ─[220Ω]─ 빨간LED(+)   │
│  GPIO 22 (15번 핀) ─[220Ω]─ 초록LED(+)   │
│  GND     (6번 핀)  ─────── 모든 GND(-)   │
└──────────────────────────────────────────┘
```

GPIO 테스트:

```bash
python3 -c "
from gpiozero import LED, Buzzer
import time
led = LED(27)
led.on(); time.sleep(0.5); led.off()
print('✅ GPIO 정상')
led.close()
"
```

## 11단계 — 자동 시작 등록 (systemd)

부팅 시 자동으로 검사 프로그램이 시작되도록 설정:

```bash
sudo nano /etc/systemd/system/inspection-edge.service
```

내용 입력:

```ini
[Unit]
Description=PCB Edge Vision Inspection
After=network-online.target
Wants=network-online.target

[Service]
Type=simple
User=pi
WorkingDirectory=/home/pi/inspection/edge
ExecStart=/home/pi/inspection/edge/.venv/bin/python main.py
Restart=on-failure
RestartSec=10
EnvironmentFile=/home/pi/inspection/edge/.env

[Install]
WantedBy=multi-user.target
```

```bash
sudo systemctl daemon-reload
sudo systemctl enable inspection-edge
sudo systemctl start inspection-edge

# 상태 확인
sudo systemctl status inspection-edge

# 실시간 로그 (Windows SSH 터미널에서 모니터링)
journalctl -u inspection-edge -f
```
