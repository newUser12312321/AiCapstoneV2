## 라즈베리파이 5 GPIO 핀맵

```
라즈베리파이 5 40핀 헤더 (실제 핀 번호 기준)

    3.3V  [01] [02]  5V
   GPIO2  [03] [04]  5V
   GPIO3  [05] [06]  GND    ← GND 공통 연결
   GPIO4  [07] [08]  GPIO14
     GND  [09] [10]  GPIO15
  GPIO17  [11] [12]  GPIO18  ← 부저 (+) 연결 (BCM 17)
  GPIO27  [13] [14]  GND
  GPIO22  [15] [16]  GPIO23  ← 초록 LED (+) 연결 (BCM 22)
    3.3V  [17] [18]  GPIO24
  GPIO10  [19] [20]  GND
   GPIO9  [21] [22]  GPIO25
  GPIO11  [23] [24]  GPIO8
     GND  [25] [26]  GPIO7
               ...

사용 핀 요약:
  11번 핀 (BCM 17) → 부저 (+)
  13번 핀 (BCM 27) → 빨간 LED (+)
  15번 핀 (BCM 22) → 초록 LED (+)
   6번 핀 (GND)   → 공통 GND
```

## 실제 배선 방법 (단계별)

### 1단계 — 부저 연결

```
라즈베리파이 11번 핀 (GPIO17) → 부저 + 단자 (긴 다리 또는 + 표시)
라즈베리파이  6번 핀 (GND)   → 부저 - 단자

※ 능동 부저는 저항 불필요
```

### 2단계 — 빨간 LED 연결

```
라즈베리파이 13번 핀 (GPIO27) → 220Ω 저항 → LED 긴 다리(+, 애노드)
라즈베리파이  6번 핀 (GND)   →              LED 짧은 다리(-, 캐소드)

저항은 LED 직렬로 연결 (어느 쪽이든 상관없음)
```

### 3단계 — 초록 LED 연결

```
라즈베리파이 15번 핀 (GPIO22) → 220Ω 저항 → LED 긴 다리(+, 애노드)
라즈베리파이  6번 핀 (GND)   →              LED 짧은 다리(-, 캐소드)
```

### 4단계 — 웹캠 연결

```
C922 USB 케이블 → 라즈베리파이 USB 3.0 포트 (파란색 포트 권장)

연결 확인:
  ssh pi@<IP>
  v4l2-ctl --list-devices
  # C922 Pro Stream Webcam ... /dev/video0 가 나오면 정상
```

## GPIO 배선 테스트

라즈베리파이 SSH 접속 후 실행:

```bash
cd ~/inspection/edge
source .venv/bin/activate

# 부저 테스트 (0.5초 울림)
python3 -c "
from gpiozero import Buzzer
import time
b = Buzzer(17)
b.on(); time.sleep(0.5); b.off()
print('✅ 부저 정상')
b.close()
"

# 빨간 LED 테스트
python3 -c "
from gpiozero import LED
import time
led = LED(27)
led.on(); time.sleep(1); led.off()
print('✅ 빨간 LED 정상')
led.close()
"

# 초록 LED 테스트
python3 -c "
from gpiozero import LED
import time
led = LED(22)
led.on(); time.sleep(1); led.off()
print('✅ 초록 LED 정상')
led.close()
"

# PASS 시나리오 시뮬레이션 (초록 LED + 짧은 비프)
python3 -c "
from hardware.gpio_controller import GpioController
gpio = GpioController()
gpio.signal_pass()
gpio.cleanup()
print('✅ PASS 신호 정상')
"

# FAIL 시나리오 시뮬레이션 (빨간 LED + 경고 비프 3회)
python3 -c "
from hardware.gpio_controller import GpioController
gpio = GpioController()
gpio.signal_fail()
gpio.cleanup()
print('✅ FAIL 신호 정상')
"
```

## 웹캠 촬영 환경 설정

### 권장 거치대 구성

```
      [웹캠]
        │  (수직 고정, PCB와 15~25cm 간격)
        │
   ─────────  ← PCB 기판 거치대 (검은 매트 위)
   [PCB 기판]

조명: 웹캠 주변 링라이트 또는 형광등 직하
      → 그림자가 생기지 않도록 균일하게 조명
```

### 초점 고정 (v4l2-ctl)

```bash
# 오토포커스 끄기
v4l2-ctl --device=/dev/video0 --set-ctrl=focus_auto=0

# 수동 초점 설정 (0~255, 15~25cm 거리는 약 30)
v4l2-ctl --device=/dev/video0 --set-ctrl=focus_absolute=30

# 현재 설정값 확인
v4l2-ctl --device=/dev/video0 --get-ctrl=focus_absolute

# 초점 미세 조정 (값을 바꿔가며 가장 선명한 값 찾기)
# 가까울수록 큰 값: 10cm≈50, 20cm≈30, 30cm≈20
```

### 프리뷰로 구도 확인

```bash
# 카메라 프리뷰 실행 (q 키로 종료)
python3 -c "
import cv2
cap = cv2.VideoCapture(0, cv2.CAP_V4L2)
cap.set(cv2.CAP_PROP_FRAME_WIDTH, 1920)
cap.set(cv2.CAP_PROP_FRAME_HEIGHT, 1080)
while True:
    ret, frame = cap.read()
    if not ret: break
    preview = cv2.resize(frame, (960, 540))
    cv2.imshow('Preview - Press Q to quit', preview)
    if cv2.waitKey(1) & 0xFF == ord('q'):
        break
cap.release()
cv2.destroyAllWindows()
"
```
